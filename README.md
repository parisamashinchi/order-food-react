<h1>Order Food by React</h1>

<p>simple web application for ordering food. </p>
<ul>
  <li>show food list</li>
  <li>show food detail</li>
  <li>order food && add to cart</li>

</ul>
<h2>Run and  build</h2>

<h5> Run it by : npm run dev </h5>
<h5> Build it by: npm run build</h5>
